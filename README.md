# Trilha Programação Python
Este repositório contém uma coleção de mini projetos e desafios práticos desenvolvidos para consolidar o domínio de programação em python.

## 🗂️ Estrutura do Projeto

**`./Estrutura_de_Dados`:** explorar métodos e caraterísticas de cada estrutura de dados (Lista, Tupla, Dicionário e Set).

**`./Controle_de_Fluxo`:** aprimorar o domínio de estruturas condicionais (if/elif/else), laços de repetição (for e while), list comprehensions e tratamento de exceções (try/except).

**`./Funcoes_e_Modulos`:** exercitar a documentação de funções, o uso de type hints e a importação, instalação e uso de pacotes externos.

**`./Series_e_Dataframes`:** entender as estruturas fundamentais da biblioteca Pandas (criação, indexação, seleção e filtragem de dados).
